# MEAN Startup Project

mongodb node js angularjs startup project.

## Installation

Use the following command to install MEAN-Startup-Project.

```bash
$ git clone https://github.com/jaydeepakbari/MEAN-Startup-Project.git
$ npm install
$ npm build
$ npm start
```
